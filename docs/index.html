Get started
Up and running in 5 minutes

Create the Slack app

Go to api.slack.com/apps, create from manifest using the included manifest.yml, and install to your workspace. Grab the App-Level Token and Bot Token.

Configure environment

Store your tokens in ~/.compass/.env:

mkdir -p ~/.compass
cat > ~/.compass/.env << 'EOF'
SLACK_APP_TOKEN=xapp-1-...
SLACK_BOT_TOKEN=xoxb-...
EOF

Run

One command. No servers, no ngrok.

bunx compass-agent

Verify

Open the app in Slack, go to the Home tab to see the dashboard, start a thread, set $cwd, and send a question.

Configuration
Environment variables
Variable Description
SLACK_APP_TOKEN Required App-level token (xapp-...) for Socket Mode
SLACK_BOT_TOKEN Required Bot user OAuth token (xoxb-...)
ALLOWED_USERS Optional Comma-separated Slack user IDs to whitelist
CLAUDE_PATH Optional Path to the claude binary (defaults to claude in PATH)
CLAUDE_ADDITIONAL_ARGS Optional Extra CLI args appended to every claude invocation
ENV_* Optional Variables prefixed with ENV_ are injected into Claude's environment
